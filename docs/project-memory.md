## Stack

- Next.js 16.2.6 App Router
- React 19.2.4
- Tailwind CSS v4
- shadcn/ui via `components.json`
- Framer Motion for reveal, stagger, hover, and tap interactions
- GSAP + ScrollTrigger for scroll progress, ticker, and parallax accents
- Static TypeScript data, no backend, no database, no Prisma

## Important Structure

- `frontend/app/page.tsx` composes all landing page sections.
- `frontend/app/layout.tsx` contains SEO metadata and Indonesian document language.
- `frontend/app/globals.css` contains the Neo Brutalism theme from `prd.md`.
- `frontend/components/sections/*` contains page sections.
- `frontend/components/shared/*` contains Navbar, Footer, BrutalCard, and SectionHeading.
- `frontend/components/shared/motion-wrapper.tsx` contains reusable Framer Motion wrappers: `Reveal`, `Stagger`, `StaggerItem`, and `MotionAnchor`.
- `frontend/components/shared/gsap-effects.tsx` registers GSAP ScrollTrigger and controls scroll progress, `[data-gsap-float]`, `[data-gsap-hero-card]`, and `[data-gsap-marquee]`.
- `frontend/components/shared/brutal-ticker.tsx` renders the animated horizontal ticker.
- `frontend/components/shared/typewriter-text.tsx` renders the hero typewriter text and respects reduced-motion preferences.
- `frontend/components/sections/timeline-section.tsx` renders the Bigboss journey timeline.
- `frontend/components/sections/price-section.tsx` renders service package cards and links CTAs to WhatsApp.
- `frontend/components/ui/*` is generated by shadcn/ui.
- `frontend/data/*` contains placeholder profile, skills, projects, and experiences.
- `frontend/data/timeline.ts` contains timeline milestones.
- `frontend/data/pricing.ts` contains pricing/service package data.
- `frontend/public/images/web-programmer.jpeg` is the user-provided image used in the hero visual.
- `frontend/public/images/projects/lynk-market-digital-store.png` is used by the Lynk Market Digital Store project card.
- `frontend/public/images/projects/bigboss-luxury-portfolio.png` is used by the Portofolio Glow Version project card.
- `frontend/public/resume.pdf` is a placeholder file and should be replaced with the real CV before publishing.

## Animation Notes

Scroll reveal animations are intentionally slower and smoother:

- `Reveal` uses Framer Motion with `useInView` and animation controls, so elements animate in when entering the viewport and animate out when leaving.
- `Stagger` also replays enter and exit behavior for grouped cards.
- Default reveal duration is around one second with a smooth cubic easing.
- Respect reduced-motion preferences before adding more intense motion.
- Hero typewriter text is configured in `frontend/components/sections/hero-section.tsx`.
